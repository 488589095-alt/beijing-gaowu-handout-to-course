# 结构骨架 + A/B/C 配方（从参考PPT「几何光学」57页一次性提炼）

> 物理是**知识点驱动**结构（非英语题型分页）。参考PPT 用途到此为止：提炼本文件后，
> 生成只复用参考PPT 的「具名版式」（品牌/版式），不克隆其成品页内容。

## 骨架
```
封面(首页)
[教材链接]（知识讲解版式 + 讲义教材链接表，可选）
每个模块(讲义 H1，如"光的折射""光的全反射")：
  章节页(Part N · 模块名)
  每个考点(讲义 H2，如"光的折射""光的色散")：
    章节小标题页(考点名 + 模块N)         ← 多考点或名异于模块时
    知识精讲页(知识讲解版式，流式分页)   ← 概念+公式+光路图
    每道典型例题(讲义 H3 下 例题N)：
      题页(典型例题版式)  题干(含inline公式)+题图+选项(文字/公式/图)
      答案页(典型例题版式) 【答案】红 + 【解析】 + （AI生成·待师审）   ← C类AI
真题呈现(章节页 + 题页→答页)            ← 讲义 H1"真题呈现"
知识总结页(知识总结版式)                ← C类AI 小结
结束页(4_自定义版式)
```

## A/B/C 来源配方
| 来源 | 判定 | 内容 |
|---|---|---|
| **A 讲义直出** | 讲义有 | 教材链接/知识精讲(概念+公式OMML+光路图)/例题题干+选项+图/真题 |
| **B 参考PPT版式** | 参考PPT有 | 品牌背景/页眉装饰/封面/章节页/结束（用具名版式，不克隆成品页）|
| **C AI生成** | 讲义无但教学需要 | **例题/真题答案+解析**（讲义全缺！）/ 知识总结小结 → 全标"待师审"+出审核清单 |

## 公式与图（物理特有铁律）
- 公式 = OMML：块公式（分式/根式/积分）→ 独立形状(a14原生+图片兜底)；
  简单行内（θ₁、n²、c>v）→ Unicode 文本进句子。**讲义公式数与成片 a14 数对账**。
- 图 = 光路图/电路图：随讲义文档锚点内联到所在知识块/例题页，**张张落页**。

## 分页规则
- 知识精讲：Flow 引擎按估高自动分页（一个考点可能多页）。
- 例题：每题题页→答页两页式；题图过大自动缩（max 5.2×2.5in）保证 4 选项同页。
