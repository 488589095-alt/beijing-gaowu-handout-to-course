# 设计 Token（从参考PPT「2026春第10讲几何光学」一次性拆解；渲染共用）

> 物理无独立模版 → 参考PPT 同时是 蓝图 + 版式来源 + token 来源。
> 拆解脚本：`extract_styles.py`（逐 run）/ `dissect_template.py`（版式）/ `blueprint_benchmark.py`（逐页）。

## 画布
- **10 × 7.5 in**（与英语/语文的 16.67×12.5 不同！所有坐标按此换算）
- 内容区：知识讲解/典型例题版式 页眉占 y0~1.15，正文 y1.3~7.15；左边距 0.5，宽 9.0

## 字体（物理 ≠ 英语）
| 用途 | 字体 |
|---|---|
| 中文标题/正文/选项 | 微软雅黑（ea） |
| 西文/数字/单位 | Times New Roman（latin） |
| 公式 | OMML 原生（PowerPoint 用 Cambria Math 渲染）|

## 颜色
| Token | 值 | 用途 |
|---|---|---|
| INK | #000000 | 正文/标题 |
| RED | #FF0000 | 强调词、答案【答案】、考点小标题前缀 |
| RED2 | #C00000 | 章节页 Part 标签 |
| AI_PURPLE | #7A2E8E | 「（AI 生成 · 待老师审核修改）」注释 |

## 字号基准（拆自参考PPT）
封面：第N讲 33 / 标题 54(讲义)~66(参考) / 年级·老师 17 ·
章节页：Part 标签 22 / 模块名 40 ·
章节小标题页：考点名 54 / 模块N 26~33 ·
页内小标题（如"光的折射率"）21.6 B ·
知识正文 15~16（概念小标题加粗）· 例题题干 16 / 选项 15 ·
答案 20 B 红 / 解析 15 · 小结 18 · 结束 33 + 40 B

| 版式名 | 用途 | 自带 |
|---|---|---|

## 版式模式
- 例题两页式：题页（题干+图+选项）→ 答案页（答案红 + 解析 + AI注释）
- 公式：每个 <m:oMath> = 独立形状 `mc:AlternateContent{Choice a14原生 + Fallback图片}`；
  简单行内公式（θ₁、c>v、sinθ₁）走 Unicode 文本进句子，不占独立形状
- 图片：光路图随锚点内联（与英语"没图别插"相反，物理图是内容核心）
